  * Banner carousel slides ([Swiper](https://swiperjs.com/get-started))

  * 下拉式選單
  * 首頁宣傳圖輪播 ([Swiper](https://swiperjs.com/get-started))
  * 燈箱效果 ([Lightbox](https://lokeshdhakar.com/projects/lightbox2/))
  * 回到網頁上方按鈕
